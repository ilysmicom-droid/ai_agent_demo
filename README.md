
# 智能代码审查 Agent 项目示例

## 项目介绍
这是一个基于 AI 驱动的智能代码审查与优化 Agent 系统示例项目，包含三个核心 Agent：代码扫描 Agent、PR 生成 Agent 和测试验证 Agent。

## 目录结构
- agents/: 各 Agent 示例代码
- utils/: 工具函数
- example_code/: 示例代码库
- requirements.txt: 依赖包

## 使用说明
1. 安装依赖：`pip install -r requirements.txt`
2. 运行 scanner_agent 扫描示例代码库
3. 运行 pr_agent 自动生成重构 PR
4. 运行 test_agent 自动执行单元测试
